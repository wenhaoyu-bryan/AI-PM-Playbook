## Prompt Engineering

### 1. Prompt Engineering for Developers

**平台**：DeepLearning.AI
**讲师**：Andrew Ng, Isa Fulford
**时长**：约 1 小时
**价格**：免费

**课程内容**：
- Prompt Engineering 的基本原则
- 如何设计有效的 Prompt
- 如何迭代优化 Prompt
- 常见的 Prompt 技巧

**适合谁**：
- 想学习 Prompt Engineering 的人
- 想提升 AI 工具使用效率的人
- 想与 AI 有效沟通的人

**关键收获**：
- Prompt Engineering 是 AI PM 的核心技能
- 好的 Prompt 需要清晰、具体、完整
- 迭代优化是提升 Prompt 质量的关键

**学习建议**：
- 边学边实践
- 用真实项目练习
- 记录有效的 Prompt 模板

## 学习建议

### 1. 带着问题学

**方法**：
- 学之前先想清楚想解决什么问题
- 学的时候带着问题找答案
- 学完后总结关键收获

**示例**：
- 问题：如何设计有效的 Prompt？
- 学习：Prompt Engineering for Developers
- 收获：Prompt 需要清晰、具体、完整

---

### 2. 边学边实践

**方法**：
- 学完一个知识点就尝试应用
- 用真实项目练习
- 记录实践中的问题和收获

**示例**：
- 知识点：如何设计有效的 Prompt
- 实践：用 Prompt 生成产品需求文档
- 收获：Prompt 需要包含具体的约束和验收标准

---

### 3. 建立知识体系

**方法**：
- 做笔记，记录关键概念和框架
- 建立知识之间的联系
- 定期复习和总结

**工具**：
- Notion：知识管理
- Obsidian：双向链接笔记
- Anki：记忆卡片

---

### 4. 加入社区

**方法**：
- 加入相关的学习社区
- 参与讨论和交流
- 分享自己的学习心得

**推荐社区**：
- DeepLearning.AI 社区
- LangChain 社区
- AI PM 社区
